Angular conventions
===================
* Create a folder for a module
* State the file type in it's name (JS only)
* use lower case with dashes between words

example:


    - app
      - my-module
        - my-service.srv.js
        - my-contoller.srv.js
        - my-directive.drv.js
        - my-filter.fltr.js
        - my-style.less
        - my-markup.html
